# nb4cut ver.1.2
                                                by Zhidao
                                                2022. 1. 3. 作成
                                                2022. 2. 6. 最終更新
--------------------------------------------------------------------
### 【説明】

光栄「信長の野望 武将風雲録」のFacedata.nb4ファイルおよびMondata.nb4
をBMPファイルに分割するツール。

--------------------------------------------------------------------
### 【コンパイル】

makeすれば実行ファイルnb4cutが出来ます。

--------------------------------------------------------------------
### 【使い方】

カレントディレクトリにFacedata.nb4とMondata.nb4を置き、
% nb4cut
とすれば、311人分の固有顔データと256人分のモンタージュデータがBMPファ
イルに出力されます。前者のファイル名はface[0-9]??*_*.bmp、後者のファ
イル名はface_mon[0-9]??.bmpとなります。[0-9]??は3桁の通し番号（開始番
号000）です。固有顔ファイルには更に名前が付け足されます。
