# nb4cut ver.1.0
                                                by Zhidao
                                                2022. 1. 3. 作成
                                                2022. 1. 3. 最終更新
--------------------------------------------------------------------
### 【説明】

光栄「信長の野望 武将風雲録」のFacedata.nb4ファイルをBMPファイルに分
割するツール。

--------------------------------------------------------------------
### 【コンパイル】

makeすれば実行ファイルnb4cutが出来ます。

--------------------------------------------------------------------
### 【使い方】

カレントディレクトリにFacedata.nb4を置き、
% nb4cut
とすれば、311人分の顔データがBMPファイルに分割されます。ファイル名に
は3桁の通し番号（開始番号000）が付きます。キャラクターによっては更に
名前が付け足されます。
